Lafe's Rules of Business

  • Never hire anyone
  • Never take investor money
  • Never willingly participate in the intellectual property system in any way
  • Never work with an individual who wants a signed contract
  • Never have any partners
  • Never ship a physical product
  • Never sell a physical product
  • Never work on weapons
  • Create work of value to all humanity
  • Release all work into public domain
  • Put all work on a public cloud storage site so it will survive if my website does not
  • Money from backers is not for services or goods; it is to patronize creation of original work
  • All business activity must be on a path to a Technological Complete Set
  • Only take money from those who can afford it
  • The three pillars of power in modern capitalism are marketing/media, finance, and military--those are the only elements that can create money from nothing
