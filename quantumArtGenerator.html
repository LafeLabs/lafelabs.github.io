Origins and Goals

The purpose of this document is to show how the Geometron meta-language can be used to construct the artistic tools required to communicate ideas about quantum computing from one human to another.

Shown below are screenshots from a electronic copy of "Quantum Computation and Quantum Information", by Nielson and Chaung, which is the closest thing quantum computing has to a definitive reference book. This is the list of symbols given in the introduction, and it's used fairly universally as a guide for graphical descriptions of quantum computation operations.

  • Illustrator doesn't know what LaTeX is and Tex doesn't know what a line is
  • tables with mathjax elements and geometron elements show power of html
  • export to svg can go to illustrator, web, powerpoint, latex, markdown, gds2, email
  • geometron engine can connect Quil or other quantum code with graphics instantly in real time, which can be manipulated to be useful also in real time
The following is a reproduction of the table from Nielson and Chuang, but with everything built in html. It is an html table, where each mathematical expression is represented with LaTeX, typeset using the Mathjax javascript library, and all the symbols are created using the Geometron geometrical language which I am advertising here.
