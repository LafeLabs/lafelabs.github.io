HTML, or "HyperText Markup Language" is the language of the web. That means on any computer, tablet, phone, or a dozeon other devices, any browser can read a file in this format, and it should look about the same. That is billions of devices in the hands of billions of people which can read your file the instant it's on a live web page! 

<!doctype html>
<html>
<head>
<title>Minimal HTML Page</title>
</head>
<body>
Here is some text, the body of the page.
</body>
</html>
