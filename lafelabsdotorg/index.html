  • Quantum Computing Funding pitch
    1. Where do computers come from?
    2. full stack must include "art layer"
    3. communication must be at the speed of Twitter or fall by the wayside
    4. HTML connects billions of devices, is totally free, everyone to everyone, every device can be a server
    5. Problem with html is organization: too many degrees of freedom, not accessible to all, not specific enough for technical communication
    6. Geometron does three things:
      1. Organize all information into data cells that have an address which corresponds to a position in a spatial model, as well as a symbol which describes what that piece of information is for.
      2. Provide a very lightweight language-independent, machine-independent and code-independent way of describing geometric/artistic figures
      3. provide a philosophical framework in which to extend itself by training more and more users and morphing into different forms as needed in other specific fields
    7. The Art Stack: code commented with symbols, code as a narrative of symbols in sequence
    8. Geometron symbol glyphs and action glyphs, structure of language
    9. How Geometron edits glyphs
    10. Practical power immediately: exporting SVG
    11. hypercube structure, roctal encoding
    12. Geometron hardware: embedding information in fab, creating code for fab, connecting fab to symbols, symbols to code, and hence code to fab
    13. Geometron as social media: connecting users in real time
    14. Leveraging the power of the open web: MathJax and other free libraries(shows that my model can work)
    15. No property, no IP, no ongoing costs, I'll help anyone to deploy, am asking for patronage in this art project
