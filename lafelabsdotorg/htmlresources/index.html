HTML External Resources and References

This page: http://www.lafelabs.org/htmlresources/
  • https://www.w3schools.com:commercial, but free and very user friendly introduction to html, css, javascript, php, and some other web tools.
  • Mozilla Developer Network: useful reference. Harder to understand than w3schools, but more complete.
  • https://codepen.ioCodepen is a free playground for trying code quickly. It's not useful for hosting a full site, and only somewhat useful for trying stuff since it's not a full page, but it's extremely useful for learning what is possible by seeing original art in html/css/javascript by talented designer/artists.
  • https://www.w3.org: the W3C Consortium, or World Wide Web consortium is the non profit group which establishes and maintains the standards that define the web such as HTML as well as many others. This page is somewhat frustrating to navigate and of limited use for learning HTML but is an important resource in predicting long term stability of software technologies which can be used on the Web.
