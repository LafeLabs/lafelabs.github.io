Introduction

The Python language is named after Monty Python the comedy group not the snake. Of all the C-like computer programming languages, I feel it is the least bad option for most general applications for several reaons. It is now becoming the standard for scientific computing, and it is easy to find useful packages which do just about everything in physics especially. In particular, the quantum computing community has pretty much completely adopted it as a standard. Also, it comes pre installed on both Macs and Linux machines and is easy to get working on almost any machine.
Python programs are stored as files ending in ".py", and those files should generally start with the so-called "shebang", which Looks like this:
